Constants for types of properties (bit maps)

 

These constants define if variables can trigger flash calculations

 

Variable

Value

Meaning

INTENSIVE_PROP

 

1

 

Intensive property (e.g. Pressure)

 

EXTENSIVE_PROP

 

2

 

Extensive property (e.g. Mole Flow)

 

CANFLASH_PROP

 

4

 

Property that can be used to trigger a flash calculation (e.g. Energy Flow)
