Useful Websites

 

www.sim42.org: The official home of the Sim42 project. You can check out the latest news and releases, report bugs, join a mailing list or download the entire Sim42 source code.

www.python.org: the source for the Python programming language

www.pfdubois.com/Numeric: the numerical libraries for Python to support matrix algebra

www.cygwin.com : the source for many UNIX tools that run under Windows, including CVS

www.wincvs.org: a Windows version of CVS

www.virtualmaterials.com: you can download the latest Virtual Materials property package system and VMGSim from Virtual Materials Group website. Look for the property package in the http://vmgsim.com/vmg/downloads?area=eos.
