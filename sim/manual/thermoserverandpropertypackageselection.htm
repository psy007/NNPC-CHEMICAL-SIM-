Thermo Server and Property Package Selection

 

While Sim42 is designed to accommodate multiple thermodynamic calculation providers, only the Virtual Materials Group property package currently provides packages to provide thermodynamic and flash calculations. This section assumes that you have installed a Virtual Materials property package system.

 

Check out www.virtualmaterials.com for an updated list of property packages for use with Sim42.

VMG Thermodynamic Models Manual (PDF format)

Before performing any process calculations you will have to start up a thermodynamic property server.

/> thermo = <Thermodynamic Method Provider>.<Desired Property Package>

For example, the command below creates the Virtual Materials thermodynamic method provider, and from it selects the SRK property package.

/> thermo = VirtualMaterials.SRK

After you have an instance of a property package running, you will add the chemical compounds of interest for calculations using the following syntax:

/> thermo + Component0 Component1 Component2 ….

For example, the command below adds a few compounds to the property package we just created:

/> thermo + Benzene Ethylbenzene Styrene

You can always inspect the content of the thermo object by inspecting it:

/> thermo
