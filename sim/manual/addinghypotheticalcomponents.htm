See the Virtual Materials Group Thermodynamic Models manual for detailed description about all the tags. The script fragment below shows how to add a hypothetical compound where the only available properties are the normal boiling point, molecular weight and liquid density at 25 C.

 

/> thermo.hypo1 = HypoCompound '''

... NormalBoilingPoint = 100 C

... MolecularWeight =  108.0

... LiquidDensity@298 = 880.0 kg/m3

... '''

 

 

Note that all physical properties necessary to define a hypothetical are enclosed using the triple quote (''') symbol.

Note that you can use the Sim42 unit conversion system when defining a hypothetical compound

 

 

Make sure you understand the physical property estimation rules used by your thermo provider.

 

 

If you are using Virtual Materials Group property package system the minimum amount of information you have to give for the estimation of a hypothetical compound physical properties is the normal boiling point.
