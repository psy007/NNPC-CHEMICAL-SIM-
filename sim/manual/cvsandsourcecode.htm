CVS and Source Code

 

Sim42 source code is maintained in a CVS repository at:

:pserver:anonymous@cvs.sim42.org/home/cvs

and can be accessed with a blank password. CVS is source version control system widely used in the open source community and available for almost all common operating systems. Windows users might consider the Cygwin environment that provides cvs along with many other Unix tools or WinCVS that provides a GUI based solution.

It is not necessary, however, to have CVS in order to browse or download the source code. A web based browser allows access from any web browser.
