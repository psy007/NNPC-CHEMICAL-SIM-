Downloads

 

These are the applications that you need to install in order to run sim42 from source code. Make sure you download the specified versions of the applications.

1.

Download and install Python 2.2.2

2.

Download and install Numerical Python 22 .

3.

Before getting sim42, you still need to get a thermodynamic server. Virtual Materials Group (VMG) provides a keyless version of their property package . Contact VMG for further thermodynamic support on their property packages.

4.

Download source code of sim42 and vmg units from CVS. This can be done from the sim42 web page, using the common version of CVS or using a gui version of CVS such as win CVS.

That is all you need in order to run sim42 but if you want to have a more complete installation of python and have access to few other goodies provided by the simulator, you should also download wxPython 2.3.3.1 and PythonWin 148 (windows users) .

 

 

If you download wxPython, you'll be able to run a more visual friendly version of the command line interface by running c:\sim\gui\python spyshell.py.

This version provides auto complete support, graphic support for manipulating files, and more...
