PropertySensor – sim.unitop.Sensor.PropertySensor

 

The property sensor detects a physical property from a port and sends a signal with the value of the physical property selected. The PropertySensor is really just a Stream_Material with a single signal port and predates the introduction of SensorPorts on streams. Using a normal Stream_Material with a SensorPort achieves the same result and is probably the preferred technique.

 

ps = Sensor.PropertySensor()
