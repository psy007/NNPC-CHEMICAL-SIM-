Create Gas-Gas Heat Exchanger

 

In this example, we use the chilled sales gas to cool the natural gas leaving the separator Sep.

/> Gas-Gas = Heater.HeatExchanger()

/> cd Gas-Gas

/Gas-Gas

/Gas-Gas> DeltaPH = 10

/Gas-Gas> DeltaPC = 10

/Gas-Gas> DeltaTHI = 10

/Gas-Gas> cd ..

 

 

Sim42 has an object-oriented design and a well-defined hierarchy of objects. You can get into the object you just created by using the "cd <object name>" command, very much like you would do if you were getting into a file folder. Conversely you move up in the object hierarchy by using the "cd .." command.

Note that Sim42 shows you where in the object level you are through the command line prompt.

Note that you can use the dereferencing operator (".") as well. For example, the commands:

/Gas-Gas> DeltaPH = 10

/> Gas-Gas.DeltaPH = 10

are identical.
