Is it really free?

 

The simulator code has been licensed using a BSD style open source license. This permits the free distribution and modification of all source code and unlike GPL style licenses does not require redistribution of modifications. We felt this was important as cases may arise where companies wish to modify the simulator for use with proprietary models. We still hope most improvements will be shared with the community.

At this time there is no open source thermo package available for the simulator. Virtual Materials Group is very supportive of this project and have made copies of their commercial property package system available for the initial development of the simulator. This package can also be downloaded and used by anyone, but unlicensed copies are restricted to the use of the RK (Redlich-Kwong) and ideal solution property packages. VMG personnel have also contributed a number of unit operations and other code to the open source repository.

Writing a high quality property package is no small undertaking, but as the thermo interface to the simulator is clearly defined, we expect that a number of property package systems, both proprietary and open source, will eventually be interfaced to the simulator. The design of the simulator even allows multiple thermo providers to be used in the same flow sheet and in some cases, such as different sides of a heat exchanger, even within one unit operation.
