Obtaining a property package engine

 

You can obtain a professional property package system from Virtual Materials Group at www.virtualmaterials.com. Virtual Materials Group actively supports Sim42 and they provide an extensive selection of property package systems for natural gas, refinery, petrochemical and chemical modeling as well as several specialized packages such as packages for urea and sulfuric acid modeling. Virtual Materials provides two free of charge property packages that can be used with Sim42 - a simple equation of state based package (RK) and an ideal solution property package. These property packages are provided without technical support.

VMG also has special programs for educational facilities which allows them to obtain professional packages inexpensively.
